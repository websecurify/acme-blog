== Description ==

WP Private Messages is a plugin for messaging each others. It requires minimum user level 0. You can send or recieve messages, and of course, reply.

For more information, please visit <a href="http://www.wpuzmani.com/">our offical page</a>

<h3>Plugin Features</h3>
<ul>
	<li>When you send messages, you can choose if you wanna save that message to your sent box.</li>
    <li>If you recieved a new message, you'll see a notice on dashboard screen</li>
    <li>You have to choose a member to send message, so I coded a simple member listing page.</li>
</ul>

== Installation ==

1. Just unzip
2. upload the "wpu-private-messages" folder to your '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Thats All! Now, you can see Private Messages menu, under "Users" panel.
